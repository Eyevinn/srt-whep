# Architecture diagram source

The animated architecture diagram embedded in the top-level `README.md`
(`docs/srt-whep-coordinator-actor.gif`) is generated by the scripts here. The
Lanshu-style renderer is hand-built (Pillow); the editable Excalidraw export is
generated alongside it.

## Files

- `render.py` — draws the diagram (dark canvas, hand-drawn wobble boxes, moving
  glow particles) and dumps 48 PNG frames. The layout is authored in a fixed
  `2520×1725` base space and supersampled by `SCALE` (default `1.5` →
  `3780×2588`); bump `SCALE` for a larger raster without touching coordinates.
  Requires Comic Sans MS (macOS ships it at `/System/Library/Fonts/Supplemental/`).
- `make_excalidraw.py` — emits `docs/srt-whep-coordinator-actor.excalidraw`, an
  editable static source mirroring the same layout (`fontFamily: 5`).
- `docs/srt-whep-coordinator-actor.excalidraw` — open in [excalidraw.com](https://excalidraw.com)
  to tweak the static layout.

## Regenerate

```bash
cd docs/diagram

# 1. static preview (fast — for tuning layout)
uv run --with pillow python render.py --png-only   # writes preview.png

# 2. animated frames, then assemble the GIF with ffmpeg (small, high quality)
uv run --with pillow python render.py              # writes frames/f000.png … f047.png
ffmpeg -y -framerate 25/2 -i frames/f%03d.png \
  -vf "palettegen=max_colors=128:stats_mode=diff" palette.png
ffmpeg -y -framerate 25/2 -i frames/f%03d.png -i palette.png \
  -lavfi "paletteuse=dither=none:diff_mode=rectangle" -loop 0 \
  ../srt-whep-coordinator-actor.gif

# 3. refresh the editable source
uv run python make_excalidraw.py                   # writes ../srt-whep-coordinator-actor.excalidraw

# 4. verify it is 3780×2588 (SCALE 1.5), 12.5fps, 48 frames, and actually animates
ffprobe -v error -select_streams v:0 -count_frames \
  -show_entries stream=width,height,r_frame_rate,nb_read_frames \
  -show_entries format=duration -of default=noprint_wrappers=1 \
  ../srt-whep-coordinator-actor.gif
```

Keep the output filename/path stable (`docs/srt-whep-coordinator-actor.gif`) so
the `README.md` embed keeps resolving. When the architecture changes, update the
box text / edges in `render.py` (and the mirrored data in `make_excalidraw.py`),
not the committed GIF by hand.
